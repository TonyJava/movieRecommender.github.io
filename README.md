# movieRecommender.github.io
• Extracted 100,000 user’s information and their movie ratings from Netflix raw data with Pig. 
• Used Hadoop MapReduce to calculate and sort the cosine similarity between users based on their ratings for overlapping movies. 
• Predicted the personal ratings of every movie for every user depending on the top N similar users’ ratings. 
• Generated the top N high rating movies for every user.
